# Coding Ninjas Website Clone (React + TailwindCSS)

This project is a **frontend clone of the Coding Ninjas website** built using **React**, **Tailwind CSS**, and **React Router**. It is designed as an **Intermediate-level project** focusing on component-based architecture, routing, responsive UI, and clean folder structure.

---

##  Live Features

* Responsive Navbar & Footer
* Home Page with Hero Section
* Courses Listing Page
* Course Detail Page (Dynamic Routing)
* Login & Signup UI Pages
* TailwindCSS Styling
* Reusable Components

---

##  Tech Stack

* **React.js**
* **Tailwind CSS**
* **React Router DOM**
* **JavaScript (ES6)**

##  Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/shrutichitkulwar-spec/coding-ninjas-clone.git
```
unzip the zip file

### 2. Install Dependencies

```bash
cd codingninja-clone-main
npm install
```

### 3. Start Development Server

```bash
npm start
```

App will run at: **[http://localhost:3000](http://localhost:3000)**

---

## Tailwind Setup

Tailwind is configured using:

* `tailwind.config.js`
* `postcss.config.js`
* `@tailwind base; @tailwind components; @tailwind utilities;`
